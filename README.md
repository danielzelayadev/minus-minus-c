# --C
Compiler for a subset of the C language developed with bison and flex.


## User Guide

### Build the Compiler

```
$ make
```

### Compile a Sample

```
$ ./compile [sample-path]
```

### Run the Compiled Sample

```
$ ./run
```

### Test a Sample

```
$ ./test [sample-path]
```

### Test All Samples

```
$ ./test
```
