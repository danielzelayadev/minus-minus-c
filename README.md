# --C
Compiler for a subset of the C language developed with bison and flex.


## User Guide

### Build the Compiler

```
$ make
```

### Compile a Sample

```
$ ./compile samples/[sample-name]
```

### Run the Compiled Sample

```
$ ./run
```

### Test a Sample

```
$ ./test samples/[sample-name]
```

### Test All Samples

```
$ ./test
```
